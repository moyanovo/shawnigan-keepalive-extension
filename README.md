## Features

- On/off toggle
- Adjustable refresh interval in minutes
- Live status display
- Match count and matched-page list
- Sends a silent background keepalive request to all open Shawnigan tabs, without reloading the UI

## Matches

- `https://shawnigan.myschoolapp.com/*`

## Load in Chrome

1. Open `chrome://extensions`
2. Turn on Developer mode
3. Click Load unpacked
4. Select `/Users/huangmoyan/shawnigan-keepalive-extension`

After that, pin the extension if you want quick access to the popup switch.
